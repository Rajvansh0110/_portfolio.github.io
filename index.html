Projects

Project 1: "Piano with Python"

Using PyQt5, a graphical user interface (GUI) is created for a piano in Python, and external libraries like Play music, threads, and wave files of the Sounds are used to produce the music. The setup of the GUI, which will contain a window for the piano keys, buttons to alter the instrument's tone and tempo, and a play/record button to record and replay the user's performance, will be the first step of the project. The computer keyboard will be used to play the piano, and each key will be programmed to make a different sound. Advanced features like exporting MIDI files, real-time note visualisation, and integration with third-party hardware like MIDI keyboards can also be added to the project. The finished product will provide customers an entertaining and engaging way to play and compose music on their computers.
